5. Groups of selectors

A comma-separated list of selectors represents the union of all elements selected by each of the individual selectors in the list. (A comma is U+002C). For example, in CSS when several selectors share the same declaratios, they may be grouped into a xomma-separated list. White space may appear beforee and/or after the comma.

In this example, we condense three rules with identical declaration into one. Thus,


h1 { font-family: san-serif}
h2 { font-family: san-serif}
h3 { font-family: san-serif}


is equivalent to:

h1, h2, h3 { font-family: sans-serif}
